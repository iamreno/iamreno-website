# Contributing

Thanks for helping improve iamreno.com.

## Ground rules
- Be respectful. No harassment, hate speech, or doxxing.
- Keep PRs small and focused.
- No secrets: do not submit API keys, passwords, private URLs, or personal data.
- Prefer plain HTML/CSS/JS unless a framework is explicitly adopted.

## How to contribute
1. Fork the repo
2. Create a feature branch
3. Open a pull request with a clear description
4. Link any related issue (if applicable)
